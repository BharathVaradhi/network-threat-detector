# Network Threat Detector

A Python-based network traffic monitoring and threat detection project.

## Features

- Packet capture using Scapy
- TCP/UDP packet analysis
- Packet metadata extraction
- CSV-based traffic storage
- Excessive request detection
- Port scan detection
- Incoming/Outgoing traffic classification

## Tech Stack

- Python
- Scapy
- CSV
- Networking Fundamentals

## Future Roadmap

- External IP analysis
- Alert engine
- FastAPI backend
- React dashboard
- ML-based anomaly detection
